# size

```cpp
std::size_t size() const;
```

## 概要
シード列の要素数を取得する。


## 戻り値
シード列の要素数を返す。


## 例外
- C++14 : 投げない


## 例
```cpp example
#include <iostream>
#include <random>

int main()
{
  std::seed_seq seq = {1, 2, 3};

  std::size_t size = seq.size();
  std::cout << size << std::endl;
}
```
