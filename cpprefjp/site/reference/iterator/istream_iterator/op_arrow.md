# operator->

* std[meta namespace]

```cpp
const T* operator->() const;
```

## 概要
イテレータを通してオブジェクトにアクセスする


## 戻り値
`&(operator*())`


## 例
```cpp example
#include <iostream>
#include <iterator>
#include <sstream>

int main()
{
  std::stringstream ss;
  ss << 1 << std::endl
     << 2 << std::endl
     << 3;

  std::istream_iterator<std::string> it(ss);
  std::istream_iterator<std::string> last;

  for (; it != last; ++it) {
    std::cout << it->c_str() << std::endl;
  }
}
```
