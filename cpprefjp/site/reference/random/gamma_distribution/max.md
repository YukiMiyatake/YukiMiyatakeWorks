# max
* random[meta header]
* std[meta namespace]
* gamma_distribution[meta class]
* function[meta id-type]

```cpp
result_type max() const;
```

## 概要
生成し得る値の上限を取得する。


## 戻り値
値の範囲の上限を返す。


## 例
```cpp example
#include <iostream>
#include <random>

int main()
{
  std::gamma_distribution<> dist(1.0, 1.0);

  double max_val = dist.max();
  std::cout << max_val << std::endl;
}
```

### 出力例
```
1.79769e+308
```
