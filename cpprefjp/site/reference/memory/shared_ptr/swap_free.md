# swap (非メンバ関数)
* memory[meta header]
* std[meta namespace]
* function template[meta id-type]
* cpp11[meta cpp]

```cpp
namespace std {
  template <class T>
  void swap(shared_ptr<T>& a, shared_ptr<T>& b) noexcept;
}
```

## 概要
2つの`shared_ptr`オブジェクトを入れ替える。


## 効果
`a.`[`swap`](swap.md)`(b)`


## 戻り値
なし


## 例外
投げない


## 例
```cpp example
#include <iostream>
#include <memory>

int main()
{
  std::shared_ptr<int> a(new int(3));
  std::shared_ptr<int> b(new int(1));

  // aとbを入れ替える
  std::swap(a, b);

  std::cout << *a << std::endl;
  std::cout << *b << std::endl;
}
```

### 出力
```

3
```
