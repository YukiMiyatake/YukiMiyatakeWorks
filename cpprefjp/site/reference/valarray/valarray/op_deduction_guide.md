# 推論補助
* valarray[meta header]
* std[meta namespace]
* valarray[meta class]
* cpp17[meta cpp]

```cpp
namespace std {
  template <class T, std::size_t cnt>
  valarray(const T(&)[cnt], std::size_t) -> valarray<T>;
}
```

## 概要
`std::valarray`クラステンプレートの型推論補助。配列と使用する要素数から推論する。


## 例
```cpp example
#include <iostream>
#include <valarray>
#include <type_traits>

int main()
{
  int ar[] = {1, 2, 3};

  // 配列と使用する要素数から推論
  std::valarray va {ar, 2};

  static_assert(std::is_same_v<
    decltype(va),
    std::valarray<int>
  >);

  for (int x : va) {
      std::cout << x << std::endl;
  }
}
```

### 出力
```
1
2
```
