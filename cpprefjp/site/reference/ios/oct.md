# oct
* ios[meta header]
* std[meta namespace]
* function[meta id-type]

```cpp
namespace std {
  ios_base& oct(ios_base& str);
}
```

## 概要
整数を八進法で出力することを指示するマニピュレータ。

## 効果
`str.setf(ios_base::oct, ios_base::basefield)`を実行する。

## 戻り値
実引数のstrオブジェクト。
