# operator/=
* chrono[meta header]
* std::chrono[meta namespace]
* duration[meta class]
* function[meta id-type]
* cpp11[meta cpp]

```cpp
duration& operator/=(const rep& rhs);
```

## 概要
現在の値を`rhs`で割る


## 効果
`rep_ /= rhs`


## 戻り値
`*this`


## 例
```cpp example
#include <iostream>
#include <chrono>

using std::chrono::duration;
using std::micro;

int main()
{
  duration<int, micro> d(9);

  d /= 2;

  std::cout << d.count() << std::endl;
}
```
* micro[link /reference/ratio/si_prefix.md]
* count()[link count.md]


### 出力
```
4
```
