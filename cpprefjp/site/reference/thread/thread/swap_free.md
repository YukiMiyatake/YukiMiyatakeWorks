```cpp
namespace std {
  void swap(thread& x, thread& y) noexcept;
}
```

## 概要
2つの`thread`オブジェクトを入れ替える


## 効果
`x.`[`swap`](swap.md)`(y);`


## 戻り値

## 例外

## 例
```cpp example
#include <thread>

int main()
{
  std::thread t1([]{ /*...*/ });
  std::thread t2;

  std::swap(t1, t2);

  t2.join();

  return 0;
}
```
