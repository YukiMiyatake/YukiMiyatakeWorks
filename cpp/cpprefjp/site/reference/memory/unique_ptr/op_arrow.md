# operator->

```cpp
pointer operator->() const noexcept;
```

## 概要
ポインタを通してオブジェクトにアクセスする。


## 要件

```cpp
get() != nullptr
```
* get()[link get.md]


## 戻り値
[`get()`](get.md)


## 例
```cpp example
#include <iostream>
#include <memory>
#include <string>

int main()
{
  std::unique_ptr<std::string> p(new std::string("hello"));

  std::cout << p->c_str() << std::endl;
}
```

### 出力
```
hello
```
