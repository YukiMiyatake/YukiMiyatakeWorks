## 概要
[`std::shared_ptr`](/reference/memory/shared_ptr.md)から`std::weak_ptr`に変換する際の、クラステンプレートの型推論補助。


## 例
```cpp example
#include <memory>
#include <type_traits>

int main()
{
  std::shared_ptr<int> sp {new int(3)};
  std::weak_ptr wp = sp;
  static_assert(std::is_same_v<
    decltype(wp),
    std::weak_ptr<int>
  >);
}
```
