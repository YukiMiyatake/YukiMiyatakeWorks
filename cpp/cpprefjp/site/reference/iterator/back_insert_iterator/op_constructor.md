```cpp
explicit back_insert_iterator(Container& x);
```

## 概要
`back_insert_iterator`オブジェクトを構築する。


## 効果
`x`へのポインタをメンバ変数`container`に保持する。

- C++11まで : `x`へのポインタは、`&x`で取得する
- C++14以降 : `x`へのポインタは、[`std::addressof`](/reference/memory/addressof.md)`(x)`で取得する
