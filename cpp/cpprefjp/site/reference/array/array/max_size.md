constexpr size_type max_size() noexcept;       // C++11

## 概要
確保可能な最大の要素数を取得する


## 戻り値
`array`クラスのテンプレートパラメータである`N`定数を返す。


## 例外
投げない


## 計算量
定数時間


## 例
```cpp example
#include <iostream>
#include <array>

int main()
{
  std::array<int, 3> ar = {1, 2, 3};

  std::cout << ar.max_size() << std::endl;
}
```
