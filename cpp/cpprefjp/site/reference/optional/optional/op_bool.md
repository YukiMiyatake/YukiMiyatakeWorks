# operator bool

* std[meta namespace]

```cpp
constexpr explicit operator bool() const noexcept;
```

## 概要
有効な値を保持しているかを判定する。


## 効果
有効値を保持している場合に`true`を返し、そうでなければ`false`を返す。


## 例
```cpp example
#include <iostream>
#include <optional>

int main()
{
  std::optional<int> p1 = 3;
  if (p1) {
    std::cout << "p1 has value" << std::endl;
  }

  std::optional<int> p2;
  if (!p2) {
    std::cout << "p2 doesn't have value" << std::endl;
  }
}
```

### 出力
