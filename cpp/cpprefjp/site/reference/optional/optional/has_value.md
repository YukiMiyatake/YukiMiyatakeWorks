# has_value

```cpp
constexpr bool has_value() const noexcept;
```

## 概要
有効な値を保持しているかを判定する。


## 効果
有効値を保持している場合に`true`を返し、そうでなければ`false`を返す。


## 例
```cpp example
#include <cassert>
#include <optional>

int main()
{
  std::optional<int> p1 = 3;
  assert(p1.has_value() == true);

  std::optional<int> p2;
  assert(p2.has_value() == false);
}
```
