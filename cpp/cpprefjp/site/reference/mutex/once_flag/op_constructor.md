```cpp
constexpr once_flag() noexcept;       // (1)
once_flag(const once_flag&) = delete; // (2)
```

## 概要
- (1) : デフォルトコンストラクタ。`once_flag`オブジェクトを初期化する。
- (2) : コピーコンストラクタ。コピー不可。

