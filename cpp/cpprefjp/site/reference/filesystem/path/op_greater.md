```cpp
namespace std::filesystem {
  bool operator>(const path& lhs, const path& rhs) noexcept;
}
```

## 概要
`path`クラスにおいて、左辺が右辺より大きいかの判定を行う。


## 戻り値
```cpp
return rhs < lhs;
```


## 例
```cpp example
#include <cassert>
#include <filesystem>

namespace fs = std::filesystem;

int main()
{
  fs::path a = "a/b/d";
  fs::path b = "a/b/c";

  assert(a > b);
}
```
