## 概要
末尾要素への参照を取得する


## 戻り値
末尾の要素への参照を返す。


## 備考
この関数の効果は、以下と同等になる：

```cpp
{
  auto tmp = end();
  --tmp;
  return *tmp;
}
```

## 例
```cpp example
#include <iostream>
#include <vector>

int main()
{
  std::vector<int> v = {3, 1, 4};

}
```
